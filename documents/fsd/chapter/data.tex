\chapter{Application Data}
\label{ch:data}

\newcounter{dataid}
\newenvironment{data}{\newcommand*\spec{\addtocounter{dataid}{10}\item[D\thedataid]}
\begin{itemize}}{\end{itemize}}

The application's main purpose is all about collecting data. This data is supposed to help to find patterns in research activity using a personal computer. Therefore there is a great range of distinctive data sets that are interesting to collect.

\section{Sessions}
\begin{data}

\spec{All sessions are stored locally in a document-based database and may be uploaded on a remote database storage. }

\spec{Each session needs to include the following data:
	\begin{itemize}
	\item Unique session id
	\item Timestamp of start and stop
	\item Video-output stream
	\item Events in chronological order
	\end{itemize}
}

\spec{Each session is contained in a mpeg-4 video container}

\spec{The video-output stream can vary but must include following data:
	\begin{itemize}
	\item Framerate
	\item Resolution
	\item Bitrate
	\item (optional) Audio
	\end{itemize}
}
\end{data}

\section{Events}

Each dataset in a session is described by an event, which has been recognized and issued by a collecting module. Each event includes multiple properties, which define and describe the event.

\begin{data}
\spec{An event must always include the following data:
	\begin{itemize}
	\item Timestamp of the occurrence
	\item Identifier of the module the event was issued by
	\item Type which identifies the events general action, e.g. a "MouseClick"
	\end{itemize}
}
\end{data}

\subsection{Window Management Events}

A window-management event is issued by the window-management module.

\begin{data}
\spec{A generic Window-Management event must include the follwing data:
	\begin{itemize}
	\item The name of the interacted window
	\item The process associated with the window
	\end{itemize}
}

\spec{A Window-Focus event should only include the general data. }

\spec{A Window-Movement event should include following additional data:
	\begin{itemize}
	\item Old and new location
	\end{itemize}
}
	
\spec{A Window-Resizing event should include following additional data:
	\begin{itemize}
	\item Old and new window size
	\end{itemize}
}
	
\spec{A Window-Maximizing/Minimizing event should include following additional data:
	\begin{itemize}
	\item The new state of the window:
		\begin{itemize}
		\item[] Maximized
		\item[] Restored
		\item[] Minimized
		\end{itemize}
	\end{itemize}
}
\end{data}

\subsection{Mouse Interaction Events}

A mouse-interaction event is issued by the mouse module.

\begin{data}
\spec{A Mouse-Click-Interaction event should include the following data:
	\begin{itemize}
	\item Clicked button:
		\begin{itemize}
		\item[] Right-Mouse-Button
		\item[] Scroll-Wheel-Button
		\item[] Left-Mouse-Button
		\end{itemize}
	\item Type which identifies if the click is a single or double click
	\item The HWND of the view, that was clicked
	\end{itemize}
}

\spec{A Mouse-Scroll-Interaction event should include the following data:
	\begin{itemize}
	\item Scroll amount
	\item The HWND of the view, that was scrolled
	\end{itemize}
}

\spec{A Mouse-Movement-Interaction event should include the following data:
	\begin{itemize}
	\item Movement vector
	\end{itemize}
}
\end{data}

\subsection{Keyboard Interaction Events}

A keyboard-interaction event is issued by the keyboard module.

\begin{data}
\spec{A Keyboard-Interaction event must include the following data:
	\begin{itemize}
	\item Tapped keys, can be single or modified keys:
		\begin{itemize}
		\item[] Ctrl
		\item[] Alt
		\item[] Alt-Gr
		\item[] Shift
		\item[] Windows
		\end{itemize}
	\end{itemize}
}
\end{data}

\subsection{Clipboard Events}

A clipboard event is issued by the clipboard module.

\begin{data}
\spec{A Clipboard-Interaction event must include the following data:
	\begin{itemize}
	\item Contained clipboard text
	\item Interaction type:
		\begin{itemize}
		\item[] Copy
		\item[] Paste
		\end{itemize}
	\end{itemize}
}
\end{data}

\subsection{Browser Events}

A browser event is issued by the browser module.

\begin{data}
\spec{A generic browser event must include the following data:
	\begin{itemize}
	\item URL of the interacted website
	\end{itemize}
}

\spec{A Navigation event must include the following additional data:
	\begin{itemize}
	\item URL of the opened website
	\end{itemize}
}
	
\spec{A Text-Input event must include the following additional data:
	\begin{itemize}
	\item Text which has been inputted by the user
	\item Textbox or form
	\end{itemize}
}
	
\spec{A Button-Tap Browser event must include the following additional data:
	\begin{itemize}
	\item Button item title
	\item URL of the linked webpage
	\end{itemize}
}
	
\spec{A Hover event must include the following additional data:
	\begin{itemize}
	\item View which has been hovered
	\end{itemize}
}
	
\spec{A Text-Selection event must include the following additional data:
	\begin{itemize}
	\item Selected Text
	\end{itemize}
}
\end{data}
