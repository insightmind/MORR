MORR ClipboardModule Testpage

This webpage contains sample cases for every event generated by the MORR ClipboardModule.
You can find a description of the expected behavior in each section.

Configuration:

  • EnabledModules must contain "MORR.Modules.Clipboard.ClipboardModule"
You can also use this preset configuration.

Test description

Each test consists of two items:
Action describes the action you (the tester) must perform in order to generate the relevant test data.
Expected behavior describes the event which should have resulted from your actions and thus be serialized into the recording file.
The following tests are grouped by event type.

In additional to the data described in the Expected behavior of each respective test, each event is also expected to hold this data:

  • TimeStamp: The serialized time of the event's occurence, e.g. "2020-03-16T17:43:52.517Z"
  • IssuingModule: "fab5bc0d-8b33-4dfd-9fa3-c58e0f1435b5"
If additional fields are stored but not mentioned, their data is not relevant to the test.

1. ClipboardCopyEvent

Test 1

Covers testcase(s): TC490

Action:
Copy the following highlighted text into your clipboard.

Expected behavior:
A new ClipboardCopyEvent is created with following data:

  • ClipboardText: "CopyMe"

2. ClipboardPasteEvent

Test 1

Important note: This test will not succeed on the current implementations of Mozilla Firefox and Google Chrome. Use Test 2 instead.

Covers testcase(s): TC490, TC500

Action:
Copy the text from the following left textfield (yellow) into the right textfield (green).

Expected behavior:
A new ClipboardPasteEvent is created with following data:

  • ClipboardText: "Copy and paste me!"

Test 2

Covers testcase(s): TC490, TC500

Action:
Copy the text from the following left textfield (yellow). Open Microsoft WordPad (e.g. by pressing WINKEY, typing "WordPad" and pressing ENTER)
and paste into empty WordPad document.

Expected behavior:
A ClipboardCopyEvent and a ClipboardPasteEvent is created, both with following data:

  • ClipboardText: "Copy and paste me!"
