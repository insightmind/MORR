MORR MouseModule Testpage

This webpage contains sample cases for every event generated by the MORR MouseModule.
You can find a description of the expected behavior in each section.

Configuration:

  • EnabledModules must contain "MORR.Modules.Mouse.MouseModule"
  • "MORR.Modules.Mouse.MouseModuleConfiguration" must exist with these values:
    • "SamplingRateInHz": 2
    • "Threshold": 200
You can also use this preset configuration.

Additional setup info

To make performing the tests easier, it is recommended (but not required) to close all unrelated windows before starting.

Test description

Each test consists of two items:
Action describes the action you (the tester) must perform in order to generate the relevant test data.
Expected behavior describes the event which should have resulted from your actions and thus be serialized into the recording file.
The following tests are grouped by event type.

In additional to the data described in the Expected behavior of each respective test, each event is also expected to hold this data:

  • TimeStamp: The serialized time of the event's occurence, e.g. "2020-03-16T17:43:52.517Z"
  • IssuingModule: "fab5bc0d-8b33-4dfd-9fa3-c58e0f1435b5"
If additional fields are stored but not mentioned, their data is not relevant to the test.

1. MouseClickEvent

Test 1

Covers testcase(s): TC440

Action:
Click the three red buttons labelled 1,2,3 in ascending order with the the left mousebutton.

Expected behavior:
Three MouseClickEvents are created with following data:

First event:

  • MouseAction: 1
  • MousePosition:
    • X: X1 (low value, close to Zero)
    • Y: Y1
    • HWnd: H

Second event:
  • MouseAction: 1
  • MousePosition:
    • X: X2 (about half your monitor's horizontal resolution)
    • Y: Y2
    • HWnd: H

Third event:
  • MouseAction: 1
  • MousePosition:
    • X: X3 (about your monitor's horizontal resolution)
    • Y: Y3
    • HWnd: H

Where X1, X2, X3, Y1, Y2, Y3 are positive integer numbers with X1 < X2 < X3 and Y1 < Y2 < Y3
and H is an arbitrary yet fixed positive integer number.




Test 2

Covers testcase(s): TC440

Action:
Rightclick the following button.

Expected behavior:
Two new MouseClickEvents are created with following data:

  • MouseAction: 2
  • MousePosition:
    • X: X1
    • Y: Y1
    • HWnd: H

Where X1, Y1 are positive integer numbers and H is a positive integer number.

Test 3

Covers testcase(s): TC450

Action:
Doubleclick (left mouse button) the following button.

Expected behavior:
Two new MouseClickEvents are created with following data:

First event:

  • MouseAction: 1
  • MousePosition:
    • X: X1
    • Y: Y1
    • HWnd: H

Second event:
  • MouseAction: 5
  • MousePosition:
    • X: X1
    • Y: Y1
    • HWnd: H

Where X1, Y1 are fixed positive integer numbers and H is a positive integer number.

2. MouseScrollEvent

Test 1

Covers testcase(s): TC460

Action:
Move your mousepointer over the following textarea and scroll down until you can see the text "done".

Expected behavior:
Several MouseScrollEvents are created with following data:

  • ScrollAmount: -120 (will vary when using a mousepad instead of a physical scrollwheel)
  • HWnd: H
Where H is a positive integer number.


Test 2

Requires Test 1 to be performed first.
Covers testcase(s): TC460

Action:
Move your mousepointer over the textarea of Test 1 and scroll upwards until you can see the text "Start scrolling here" again.

Expected behavior:
Several MouseScrollEvents are created with following data:

  • ScrollAmount: 120 (will vary when using a mousepad instead of a physical scrollwheel)
  • HWnd: H
Where H is a positive integer number.

3. MouseMoveEvent

Test 1

Covers testcase(s): TC470

Action:
Move your Mousepointer over the following red buttons labelled 1,2,3 in ascending order. Try to move along a straight line when doing so.

Expected behavior:
Multiple MouseMoveEvents are created with following data:
Event i:

  • MousePosition:
    • X: X(i)
    • Y: Y(i)
  • HWnd: H
Where H is a positive integer number and X(i) < X(i+1) and Y(i) < Y(i+1) for all events i.
