MORR Webextension Testpage

This webpage contains sample cases for every event generated by the MORR webextension.
You can find a description of the expected behavior in each section.
When you see [TestRoot] in the data description of an event, this refers to the root-path of the testpage, e.g. file:///C:/Users/JohnDoe/Documents/IntegrationTestPage/

1. ButtonClickEvent

Test 1

Action:
Click the following button

Expected behavior:
A new ButtonClickEvent is created with following data:
ButtonTitle: "Click Me!"
ButtonHref: null

Test 2

Action:
Click the following highlightet text.

Expected behavior:
A new ButtonClickEvent is created with following data:
ButtonTitle: "Click Me (Href)"
ButtonHref: "SubPages/HrefToHere.html"

Click Me (Href)

2. CloseTabEvent

Test 1

Action:
Click the following highlightet text to open a new Tab. Close this new tab again to return to the current page.

Expected behavior:
A new OpenTabEvent, two SwitchTabEvents, and a new CloseTabEvent are created. The CloseTabEvent contains following data:
CurrentURL: "[TestRoot]Subpages/CloseTabSubPage.html"

Click Me to open a New Tab

3. FileDownloadEvent

Test 1

Action:
Download the following image, e.g. by rightclicking the highlighted text and choosing "Save Link as".

Expected behavior:
A new FileDownloadEvent is created with following data:
URL: "[TestRoot]Subpages/download.png"
MIMEType: "image/png"

Download

4. HoverEvent

Test 1

Action:
Move your mousepointer over the following element and leave it there for 0.5 seconds.

Expected behavior:
A new HoverEvent is created with following data:
HoveredElement: "Hover Me!"

5. NavigationEvent

Test 1

Action:
Click the following highlighted text to navigate to another URL.

Expected behavior:
A new NavigationEvent is created with following data:
CurrentURL: "[TestRoot]SubPages/HrefToHere.html"

Click Me to navigate

6. OpenTabEvent

Test 1

Action:
Click the following highlightet text to open a new Tab.

Expected behavior:
A new OpenTabEvent is created with following data:
CurrentURL: "url:unknown"

Click Me to open a New Tab

7. SwitchTabEvent

Test 1

Action:
Click the following highlightet text to open a new Tab. Afterwards, switch to the previous Tab (containing this webpage).

Expected behavior:
Two SwitchTabEvents are created.
The first SwitchTabEvent has this data: CurrentURL: "[TestRoot]IntegrationTestPage.html"
The second SwitchTabEvent has this data: CurrentURL: "[TestRoot]SubPages/SwitchTabSubPage.html"

Click Me to open a New Tab

8. TextInputEvent

Test 1

Action:
Enter any text (subsequently called W) in the following input textfield and then remove focus from the input field (e.g. by pressing Tab or leftclicking to the right of the textfield)
Note: The textfield is limited to 8 characters to keep the test short.

Expected behavior:
A new TextInputEvent is created with following data:
InputtedText: W = your text
Textbox: "TestTextField"

9. TextSelectionEvent

Test 1

Action:
Select any text from the following sentence by pressing and holding the left mouse button at the start of your selection and releasing the left mouse button at the end of your selection.
Your selected text is subsequently called W

Expected behavior:
A new TextSelectionEvent is created with following data:
SelectedText: W = your selected text

Select from this sentence:
In almost every program, text can be selected using either the mouse or the keyboard.

Test 2

Action:
Select text like in test 1, but do at least leave out the last word and select your text from left to right (otherwise the next step may shrink the selection instead of increasing it).
Your selected text is subsequently called W
Next, add to the selected range by pressing and holding the shift-key and pressing the RIGHTARROW key two times.

Expected behavior:
Three new TextSelectionEvents are created with following data:
First event:
SelectedText: W = your selected text Second event:
SelectedText: W + 1 character to the right Third event:
SelectedText: W + 2 characters to the right

Select from this sentence:
In almost every program, text can be selected using either the mouse or the keyboard.
